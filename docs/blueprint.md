# **App Name**: InfoFlow Hub

## Core Features:

- Content Creation: Admin tool to create and format articles with a rich text editor. Access is restricted to admins via a separate URL.
- Article Publishing: Publish created articles to the main website, organized by category or tag.
- Layout Customization Tool: LLM tool to optimize content blocks based on analysis of similar sites and an explicit user input for type of the new site to build, and target audience to maximize engagement.
- Homepage Layout: Display featured articles, recent posts, and categorized content sections.
- Admin Access Control: Secure admin access via a specific URL, separate from the user-facing website, ensuring content management is restricted.
- Category Management: Allows admins to create, edit, and delete categories for organizing articles.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey expertise and sophistication.
- Background color: Light gray (#F5F5F5), near-white but gentler on the eyes, maintaining a clean backdrop for content.
- Accent color: Electric purple (#8E2DE2), slightly desaturated to provide emphasis to buttons and active elements without overwhelming the user. (#8E2DE2)
- Clean, grid-based layout for easy content discovery, inspired by Lifehacker.
- Body text: 'Literata', a serif font for readability and a classic feel. Headline: 'Space Grotesk', a sans-serif font to ensure contemporary, precise feel
- Simple, consistent icons for categories and navigation.
